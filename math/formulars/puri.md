# PURI

Alle Formeln im Überblick:
* $U$
    * $U = R * I$ - URI
    * $U = \frac{P}{I}$ - UPI
    * $U = \sqrt{P * R}$ - UPR
* $R$
    * $R = \frac{U}{I}$ - RUI
    * $R = \frac{P}{I^2}$ - RPI
    * $R = \frac{U^2}{P}$ - RUP
* $I$
    * $I = \frac{U}{R}$ - IUR
    * $I = \sqrt{\frac{P}{R}}$ - IPR
    * $I = \frac{P}{U}$ - IPU
* $P$
    * $P = U * I$ - PUI
    * $P = I^2 * R$ - PIR
    * $P = \frac{U^2}{R}$ - PUR
