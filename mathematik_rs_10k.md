## Typen und Bezeichnungen

### Das `'` Zeichen

* **Winkelminuten**: Ein einzelnes Hochkomma (Prime) hinter einer Zahl wird verwendet, um Winkelminuten anzugeben.
Beispiel: $41°15'$ bedeutet 41 Grad und 15 Minuten.
* **Ableitung (Prime-Notation)**: In der Analysis wird ein einfaches Anführungszeichen, oft als "Prime" bezeichnet (z.B. $f'$), verwendet, um die Ableitung einer Funktio (z.B. $f(x)$) zu kennzeichnen.
* Unterscheidung/Indexierung: Es wird genutzt, um verwandte Variablen zu unterscheiden, zum Beispiel $x$ und $x'$ ("x-Strich").

## Seiten- und Winkelberechnungen am Dreieck

### Satz des Pythagoras
$c^2 = a^2 + b^2$

### Winkelbeziehungen

$sin \alpha = \frac a c = \frac {Gegenkathete-von-\alpha }  {Hypotenuse}$

$cos \alpha = \frac b c = \frac {Ankathete-von-\alpha} {Hypotenuse}$

$tan \alpha = \frac a b = \frac {Gegenkathete von \alpha} {Ankathete-von-\alpha}$

### Sinussatz

$\frac a {sin \alpha}$

$\frac b {sin \beta}$

$\frac c {sin \gamma}$

### Kosinussatz

$a^2 = b^2 + c^2 - 2 * b * c * cos\alpha$

$b^2 = a^2 + c^2 - 2 * a * c * cos\beta$

$c^2 = a^2 + b^2 - 2 * a * b * cos\gamma$

### Ähnlichkeitsbeziehungen

Wenn zwei Dreiecke in ihren Winkeln übereinstimmen $\alpha = \alpha'$, $\beta = \beta'$ und $\gamma = \gamma'$

Für die Längenverhältnisse gilt:

$\frac a b = \frac {a'} {b'}$

$\frac a c = \frac {a'} {c'}$

$\frac b c = \frac {b'} {c'}$
